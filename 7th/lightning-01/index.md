# Kernel report
제7회 한국 리눅스 커널 개발자 모임 발표: 라이트닝 토크

## 발표내용
대략 2달에서 3달 간격으로 mainline linux kernel이 릴리즈되고 있다.
최근에는 과연 어떤 내용들이 linux kernel에 반영되고 있는지 보기위해,
linux 5.4부터 linux 5.18까지의 주요 변경사항을 정리해서 공유할 예정이다.

## 발표자 소개
