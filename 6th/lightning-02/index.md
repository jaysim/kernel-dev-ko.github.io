# BPF 소개 및 적용 사례
제6회 한국 리눅스 커널 개발자 모임 발표: 라이트닝 토크

## 발표내용
리눅스 커널의 BPF infrastructure를 간략히 소개하고 커널에서 BPF가 어떻게 활용되고 있는지 다양한 사례(Tracing, XDP, IR decoding, overriding return value 등)를 소개한다.

## 발표자 소개

### 송태웅
리눅스 Tracing 기술 관련 오픈소스 활동을 하고 있고, 리얼리눅스라는 
리눅스 교육업체를 운영하고 있다.
